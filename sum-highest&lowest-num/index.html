Sum sum-without-highest-and-lowest-number

  sum all the numbers of a give array except the highest and the lowest elements (by value , not by index)
      the highest or lowest element respectively is a single element at each edge even if there are more than one with the same value 
      mind the input validation 
      Example 
       {6, 2, 1, 0, 10} => 16
       {1, 1, 11, 2, 3} => 6
